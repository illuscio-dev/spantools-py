classmethod is_mimetype(value, mimetype)[source]¶

Checks if string or enum is equivalent to a given mimetype enum value. Similar to isinstance() or issubclass().

Parameters

  • value (Union[MimeType, str, None]) – to check

  • mimetype (MimeType) – to check against.

Return type

bool

Returns

True: is equivalent. False: is not Equivalent.

Does some cleaning (such as resolving case and removing ‘x-‘ from type).

For instance, all of the following will pass equivalence for MimeType.YAML:

  • ‘application/yaml’

  • ‘application/YAML’

  • ‘application/x-yaml’

  • ‘yaml’

  • ‘x-yaml’

encode_content(content, mimetype=None, headers=None, data_schema=None, validate=False, encoders=None)[source]¶

Encodes content object to bytes using encoder / schema.

Parameters

  • content (Optional[Any]) – Object to be encoded.

  • mimetype (Union[MimeType, str, None]) – Content-Type to serialize to.

  • headers (Optional[MutableMapping[str, str]]) – Request headers to which content information should be added.

  • data_schema (Optional[Schema]) – Marshmallow schema. Will be used to dump / validate content before encoding.

  • validate (bool) – Whether to validate content after dumping. This may come with a big performance penalty.

  • encoders (Optional[Mapping[Union[MimeType, str, None], Callable[[Any], bytes]]]) – Custom set of encoders to use for encoding content.

Return type

bytes

Returns

Encoded content for request.

Raises

  • ContentTypeUnknownError – If a method for serializing / validating the content is unknown and the content is not already bytes.

  • ContentEncodeError – If error occurs when encoding content.

  • marshmallow.ValidationError – If raised while dumping / validating content via data_schema.

decode_content(content, mimetype=None, data_schema=None, allow_sniff=False, decoders=None)[source]¶

Loads content by decoder / schema from received mimetype.

Parameters

  • content (bytes) – Received binary body.

  • mimetype (Union[MimeType, str, None]) – mimetype info if known.

  • data_schema (Optional[Schema]) – marshmallow schema to use to load data to model / object

  • allow_sniff (bool) – If mimetype is unavailable, whether to attempt to load content anyway.

  • decoders (Optional[Mapping[Union[MimeType, str, None], Callable[[bytes], Any]]]) – Custom set of decoders to use.

Return type

Tuple[Optional[Any], Optional[Any]]

Returns

(loaded data object, loaded mimetype) tuple.

Raises

  • ContentTypeUnknownError – If a method for decoding / validating the content is unknown or unregistered, and allow_sniff is False.

  • ContentDecodeError – If error occurs when decoding content or no registered. decoders succeed if allow_sniff is True

  • NoContentError – If no content is passed to be decoded. This error is inherited from ContentDecodeError, so catching ContentDecodeError also catches NoContentError.

  • marshmallow.ValidationError – If raised while loading content via data_schema.

API Exceptions¶

APIError and exceptions that inherit from it are designed to be raised by spanserver during the processing of a request, then transmitted back for handling.

API exceptions are found in this toolbox for libraries which wish to consume these errors without installing spanserver and all of its dependencies.

These errors can be found in the spantools.errors_api submodule.

exception APIError(message, error_data=None, error_id=None, send_media=False)[source]¶

Base error returned by SpanAPI in http responses. Must have a unique code.

http code: 501

api code: 1000

Returned when any exception not inherited from APIError is raised from a SpanRoute method.

Init Values / Attributes

  • message: str: error message passed back in response

  • error_data: Optional[dict]: error data to be passed back in response

  • send_data: bool: send regular response data as well as error. If False: 'data' object of response is set to None
